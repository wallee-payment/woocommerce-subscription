1Prerequisites

If you don’t already have one, create a wallee account.

2Installation

  1. Install it directly from your Wordpress Installation under Plugins > Add New search for wallee Subscription.

  2. The plugin will now be visible under Plugins. You can install the plugin directly from here. There is no further configuration required.

3FAQ

3.1How can I translate the plugin?

  1. Install the Wordpress Loco Translate plugin.

  2. Go to Loco Translate > Plugins.

  3. Click on WooCommere wallee Subscription.

    1. Click on the existing language and adapt the translations.

    2. Cick New Langugage and translate the strings.

4Support

If you need help, feel free to contact our support.

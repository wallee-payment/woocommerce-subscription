== Installation ==

= Minimum Requirements =

* PHP version 5.6 or greater
* WordPress 4.4 or greater
* WooCommerce 3.0.0 or greater
* wallee Plugin 1.2.4 or later
* Woocommerce Subscriptions Plugin 2.5 or later

= Automatic installation =

1. Install the plugin via Plugins -> New plugin. Search for 'wallee Subscription '.
2. Activate the 'wallee Subscription' plugin through the 'Plugins' menu in WordPress
3. You're done. You can now process subscriptions through wallee.

= Manual installation =

1. Unpack the downloaded package.
2. Upload the directory 'woo-wallee-subscription' to the `/wp-content/plugins/` directory
3. Activate the 'wallee Subscription' plugin through the 'Plugins' menu in WordPress
4. You're done. You can now process subscriptions through wallee.
